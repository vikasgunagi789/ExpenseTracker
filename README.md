# Smart Expense Tracker 

This project is made for the Advance Java Course Mini Project 

A mini project developed using Advanced Java concepts such as JSP, Servlets, JDBC, and MySQL.

##  Features
- User Registration
- User Login
- Database Connectivity
- Expense Tracking
- Savings Calculation
- JSP and Servlet Integration

##  Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- Eclipse IDE

## Database
Database Name:
expense_tracker

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat Server
3. Add MySQL Connector JAR
4. Create MySQL database
5. Run project on server

- Home Page
- Registration Page
- Login Page
- Database Output
